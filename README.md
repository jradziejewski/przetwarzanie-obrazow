# [PL] Przetwarzanie obrazow

Jest to projekt na studia w języku C.<br /> 
Pozwala na zaimportowanie obrazu w formacie pgm, który jest zczytywany i zapisywany do dwuwymiarowej tablicy a następnie można na nim wykonać następujące przekształcenia:
<br />
- obrót, <br />
- efekt negatywu, <br />
- generowanie i filtrowanie szumu, <br />
- wygenerowanie histogramu <br />.
- możliwe jest wczytanie kilku obrazów i przechowywanie bazy.

# [EN] Image processing

This is a project for a university class in C. <br />
It allows you to import an image in PGM format, which is read and saved to a two-dimensional array, and then you can perform the following transformations:
<br />
- rotate, <br />
- negative effect, <br />
- noise generation and filtering, <br />
- generating a histogram <br />.
- user can upload several images that are stored in a database.
